# DCIT 208 Client Project

## Team Name
Apex Unified

## Project Title
HavenRae Project Management Platform

## Client
Angel Sharon

## Problem Statement
HavenRae Studios, an interior design company specializing in furnishing and styling residential rental units, is growing rapidly but managing operations manually across multiple platforms. This has fragmented project information and reduced productivity.
According to Operations Manager Angel Sharon, the company struggles to track progress across multiple units, manage timelines, monitor furniture and décor procurement, control budgets, and meet deadlines — largely due to the lack of a centralized system providing real-time visibility.
The proposed solution is a centralized project management platform covering scheduling, task management, budget tracking, supplier management, and documentation — improving coordination, reducing administrative workload, and supporting the company's continued growth.
